# Python Programming Showcase

Welcome! This repository serves as a showcase of my Python programming capabilities, built during my journey through Harvard University's CS50P course. It features modular, tested code handling data pipelines, graphics manipulation, and input validations.

---

## 🚀 Project Highlights

### 1. Data Cleaning Pipeline (`scourgify.py`)
* **What it does:** Reads a raw CSV data sheet containing poorly organized strings, cleans up structural naming formats, and generates a newly structured data output file.
* **Skills demonstrated:** File I/O operations, structural CSV parsing using dictionaries (`DictReader`/`DictWriter`), and data sanitization.

### 2. Graphical Overlay Utility (`shirt.py`)
* **What it does:** Automates image manipulation by parsing terminal-level file names, checking background layer configurations, and overlaying template apparel graphics perfectly onto an input target.
* **Skills demonstrated:** System arguments parsing (`sys.argv`), graphics processing with the `Pillow` (PIL) library, and custom error boundaries (`FileNotFoundError`).

### 3. Conditional Validator (`plates.py`)
* **What it does:** Enforces a multi-layered rule matrix to validate custom sequential strings (Vanity Plates) based on size constraints, character distribution layout, and absolute digit requirements.
* **Skills demonstrated:** Complex boolean logic, loop iterations, and strict string formatting rules.
